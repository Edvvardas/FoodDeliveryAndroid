# Food Delivery App (Android)

This is an Android food delivery app project for a course assignment. The app includes user registration/login, restaurant browsing, menus, ordering, order tracking, and reviews, with separate flows for customers and drivers.

## What?s in this repository
- Android client application (UI, adapters, models, activities)
- Gradle build configuration

## What?s not included
- The Spring Boot backend is **not** included in this repository.
